Install penguin.js

Start building with penguin.js

getting started penguin.js is easy because it embraces the filesystem as its API. So there little need to fiddle in config files to just get-started out with your new project.

github.com/domachine/penguin.js


Create a package.json with minimal markup

{
  "penguin": {
    "languages": ["en"]
  },
  "scripts": {
    "serve": "penguin serve",
    "prebuild": "penguin pack",
    "build": "penguin build"
  }
}

Install penguin.js (make sure, you have at least node v6 installed)

npm install -S penguin.js

Start your development server

npm run serve

Alright now if you got everything right, your development server should start and listen on port 3000.

Create pages/whatever.html file and add some content. This will be served under /whatever.

To implement inplace editing within seconds check Getting Started.
